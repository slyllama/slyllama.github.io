Jade Spring 1.2b

Steam Beta 1.2b

Hello everyone! First of all, thank you so much everyone for playing and enjoying Jade Spring; I’ve been so happy to hear people’s comments and ideas since the game came out on Steam on the 13th. OMG, Mukluk covered the game (you can watch it here), I couldn’t believe it. And it also got a play-through and review from AngryZorric. I’ve got a bunch of decorations that I’ve started, and all this makes me really excited to finish them and get them all in the game. Thanks again, from my heart, ahhh.

For now though, here’s the 1.2b version of Jade Spring ! There are a couple of new features but it’s mainly comprised of lots and lots of bug fixes. I’m hoping to make the game really stable over the next few versions while I continue to work on new decorations, and then put out a release version with a plenitude of new stuff. Here’s a list of everything that’s been changed and fixed:

Changelog (1.2b)

Note: the Jade Spring 1.2b version will be a bit bigger than it has been in the past (about 130MB). This is because the engine that the game uses (Godot) has also updated, and I’ve just moved Jade Spring up to that new version.

  • Quickly find walls, floors, etc with category tiles in the decoration pane.
  • Fixed several instances of stuttering on load (transition to Godot 4.4.1).
  • An effect will display when Ratchet has optional dialogue.
    • Contributes to the ‘My Friend Ratchet’ achievement.
  • Added a banner which will turn you into a miniature Jade Bot!
  • Added ‘String Lights’ decoration.
  • Added an option to clear the shader cache in settings.
  • Directing the camera upward or downward will guide the Jade Bot to ascend or descend.
  • ‘Enter’ and ‘space’ keys can now be freely bound by the player without conflicts.
  • Directional movement keys can be changed to control the rotation of the camera rather than strafing.
  • Fixed skill button conflicts where skills could be used simultaneously or while manipulating decorations.

Minor bugfixes and improvements

  • Fixed the camera spinning out of control at low FPS.
  • Fixed an issue where the sun would no longer be rotatable after returning to the main menu.
  • Fixed the menu art flickering on load.
  • Fixed the Slyllama website button spawning multiple browsers.
  • Fixed the appearances of some text and dialogue boxes.
  • Fixed an issue where the window mode might not update properly in settings.
  • Fixed an issue where selecting multiple decorations might miss some decorations.
  • Attenuator marker is now properly aligned to tracks.
  • Reduced the intensity of Jade Bot engine heat distortion.
  • Fixed Karma eligibility not updating properly in decoration pane.
  • ‘Warp to safe point’ button no longer opens screenshot folder on right click.
  • Updated credits.
  • Various small wording and bug fixes.
