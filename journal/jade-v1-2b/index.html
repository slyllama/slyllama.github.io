DRAFT

Jade Spring 1.2b

Note: the Jade Spring 1.2b will be a bit bigger than it has been in the past (about 130MB). This is because the engine that the game uses (Godot) has also updated, and I’ve just moved Jade Spring up to that new version.

1.2b Changelog

  • Quickly find walls, floors, etc with category tiles in the decoration pane.
  • Fixed several instances of stuttering on load (transition to Godot 4.4.1).
  • Fixed an issue where the run would no longer be rotatable after returning to the main menu.
  • Fixed the appearances of some text and dialogue boxes.
  • Fixed an issue where the window mode might not update properly in settings.

1.1b Changelog

(See the full article.)

Decorations

  • Added the ability to duplicate and move multiple decorations at once.
  • Added noodle bar stool.
  • Added neon noodle store display.
  • Added a small clump of grass.
  • Lanterns now emit light.

Saves and game versions

  • Loading a new decoration save file with an older version of Jade Spring will generate a warning.
  • Save file backups are now generated for each new version of Jade Spring.
  • A small indicator will show on the main menu if there is a newer version of the game that you can download.

Fixes and minor improvements

  • Many small script fixes and improvements, with a big thanks to areyoshi.
  • Light rays no longer appear at night.
  • Fixed extra clicking sounds when applying transformations.
  • Fixed snapping persisting after changing from decoration rotation to movement.
  • Lights now show a hint as to how their intensity and color can be modified.
  • Light previews no longer disappear during editing after being duplicated.
