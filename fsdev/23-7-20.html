More FSX I/O – 23/7

About a week ago I began to investigate outputs and inputs to and from FSX, and after a lot of (sometimes frustrating) searching, I managed to solicit help and get access to the tools that I needed! Now we are beginning to make some progress – as I've recorded in this video, Python is successfully picking up throttle data (albeit using the wrong conversion type – I’ll sort that out later :P).

The next step will be to actually write and read data, starting first through Python, and then using TCP or UDP to communicate with the Pi.
